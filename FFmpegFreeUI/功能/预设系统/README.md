# 预设系统

`预设数据_v6.vb` 只定义可序列化的预设模型及其嵌套数据结构。

`开发者内置预设_v6.vb` 维护内置预设清单和预设克隆逻辑。

`预设存储_v6.vb` 负责预设文件路径、JSON 读写和旧数据初始化。

`预设滤镜排序_v6.vb` 负责内置滤镜顺序、旧字段迁移及滤镜排序同步。

`预设面板总览_v6.vb` 负责面板级保存、加载、重置和参数总览。

`预设面板映射_v6.vb` 集中维护预设模型与参数面板控件的双向映射。

`预设命令行核心_v6.vb` 协调阶段化命令行生成；其余 `预设命令行_*.vb` 按滤镜图、编码、输入输出、通用工具和滤镜构造拆分实现细节。

所有命令行与面板实现仍是 `预设管理_v6` 的 Partial 部分，外部调用的类型和方法签名保持不变。
